## Features

* device music play
* background play
* favorite artists
* favorite songs
* playlists
* shortcuts


## Environments

### Flutter

* Flutter
* Dart
* Visual Studio Code

### Android

* Android Studio Koala | 2024.1.1
* Kotlin
* minSdkVersion 33
* targetSdkVersion 34
* Google Pixel 4a (Android Version 13)
* Google Pixel 7 Pro (Android Version 14)

### iOS

* Xcode 15.4
* Swift
* iOS Deployment Target 15
* iPhone 12 mini (iOS 16)
* iPhone 14 Plus (iOS 17)
